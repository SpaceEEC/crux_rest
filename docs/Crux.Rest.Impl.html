    Crux.Rest.Impl (crux_rest v0.3.0-dev) View Source

    A module providing the functions behind Crux.Rest.

    All functions return a Crux.Rest.Request struct, that can be executed using Crux.Rest.request/1 or Crux.Rest.request!/1.

    For documentation about these functions see Crux.Rest.
